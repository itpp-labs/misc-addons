Read Only User

You can set any user to readonly except administrator.

Go to setting --> Users --> Users --> Select User --> Click on Set Read Only button

Key features:
  • Easy To Use
  • User Wise Setting
  • Set/Unset Any User To Read Only.
Click on Set Read Only button to set selected user to Read Only. Depend on access right set for selected user user can read any menus/views.
Click on Unset Read Only button to reset read only behaviour.
Before set user to Read Only, user can able to Create/Edit/Delete/Duplicate record.
After set user to Read Only, user Create/Edit/Delete/Duplicate options are not visible. If user click on any button System will raise access denied warning.
If user click on button that open another view or wizard then system will allow to do this thing but there should not any transaction in background.
If user click on button that create any record or prcess some transaction then System will raise an Assess Error.
