Multiple SO to one invoice

Put the lines from multiple sale orders to one invoice

The module adds the new 'Sales Orders Lines' menu and allows to create one invoice from lines of multiple sale orders.

Go to Sales → Sales Orders Lines and select the lines from different sale orders. Then click on the 'More' droplist and select the 'Create grouped Invoice' item.

As a result you can see new invoice that contains the lines you have selected. It should be noted that you can group the lines of sale orders from the same customer only, otherwise invoices will be created per each customer separately.
