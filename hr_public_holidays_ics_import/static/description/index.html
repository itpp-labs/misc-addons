Public Holidays Import ICS

Import holidays from .ics file

The module allows to import public holidays from .ics file

How it works

Go to Human Resources → Public Holidays menu and open or create new item. Click on the "Import from ics" button.

In the form that appears select a file you need to import and click on the "Import" button.

Then you can see the holidays successfully imported.
