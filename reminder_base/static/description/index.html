Reminders and Agenda (technical core)

Organise all your tasks, todos, events, etc.

The module provide easy way to add notifications to any form with date field.

For any form with non-empty date field related calendar event is created. It allows to check your agenda in one place.

From event form you can easily open related record.

After installing modules, you can create events for existed records via tool in Settings\Calendar (for new or updated records events are created\updated automatically).

You have to install additional modules for each type of record you need. Search Reminders and Agenda in modules list.

  • Reminders and Agenda for Opportunities
  • Reminders and Agenda for Tasks
  • Reminders and Agenda for Phonecalls
  • ...
