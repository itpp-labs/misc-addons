Autostaging Project Task

Change stages of tasks automatically after a specified time

Usage

  • Go to Project → Configuration → Stages
  • Choose a stage which position you want to change automatically after an established time
  • Mark the "Autostaging enabled" box
  • Select the necessary next stage
  • Specify "Autostaging idle timeout" (in days) - the time after which a task should be moved

E.g. after 1 day the tasks from the Basic stage will be moved to the Advanced automatically
