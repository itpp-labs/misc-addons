Project Task Order

Order tasks by kanban state: Normal, Done, Blocked

The module allows to order tasks by "kanban state" values:

  1. Normal
  2. Done
  3. Blocked
It helps easier to monitor the existing tasks.

You can see how tasks are ordered in the "Project → Tasks" menu. Firstly, you will see the tasks which have the "normal" state, below it will be tasks with the "done" state. At the last you will see the blocked tasks.
